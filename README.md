# 掘金一下

掘金一下是一个专注于管理你的掘金文章的 VS Code 扩展，支持以下功能：

- [x] 右下角点击悠悠在浏览器中打开掘金
- [x] 查看自己已发布的文章列表
- [x] 根据标题匹配搜索自己的文章
- [ ] 根据标题和内容模糊匹配搜索
- [ ] 查看草稿文章列表
- [ ] 在 vscode 中管理自己的文章

> 打开掘金一下请认准：`Juejin Me: start`
