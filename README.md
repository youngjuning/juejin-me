# 掘金一下

掘金一下是一个专注于管理你的掘金文章的 VS Code 扩展，支持以下功能：

- 右下角点击悠悠在浏览器中打开掘金
- 查看自己已发布的文章列表
- 根据标题模糊搜索
- 根据简介模糊搜索
- 模糊匹配支持使用正则，比如 `^React`

## 命令

- 打开掘金一下请认准：`Juejin Me: start`
- 设置掘金用户 ID 请认准：`Juejin Me: Config User Id`
